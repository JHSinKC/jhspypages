'flask' Web Micro-Framework
Version: 1.1.1 Released: Jul 2019 Packaged with (5) dependencies
Flask is a lightweight WSGI web application framework. It is designed to make getting started quick and easy, with the ability to scale up to complex applications.
Flask doesn’t enforce any dependencies or project layout, but it does come packaged with five other apps (see below).

    'flask-login' Module
    Version: 0.4.1 Released: Dec 2017 no warnings
    Flask-Login provides user session management (common tasks of logging in, logging out, and remembering your users’ sessions over extended periods of time). Flask-Login is not bound to any database system or permissions model. The only requirement is that your user objects implement a few methods, and that you provide a callback to the extension capable of loading users from their ID.
