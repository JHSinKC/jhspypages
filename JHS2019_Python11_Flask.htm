    'flask-login' Module
    Version: 0.4.1 Released: Dec 2017 no warnings
    Flask-Login provides user session management (common tasks of logging in, logging out, and remembering your usersâ€™ sessions over extended periods of time). Flask-Login is not bound to any database system or permissions model. The only requirement is that your user objects implement a few methods, and that you provide a callback to the extension capable of loading users from their ID.
